DBPackage

Pacote de classe nativo do PKD que dá uma maior facilidade em manipulação de banco de dados.

  • Conectando com o banco de dados
    Drive::set('mysql');
    $conn = Drive::getConnection(string $host, string $dbname, string $user, string $pass);
    $conn->close();
  • Realizando consulta de forma direta 
    Drive::set('mysql');
$conn = Drive::getConnection(string $host, string $dbname, string $user, string $pass);

$stmt = $conn->prepareStatement("INSTUÇÃO SQL");
$stmt->execute([array $values]);

$conn->close();
  • A classe SqlInsert, para inserir dados 
    Drive::set('mysql');
$conn = Drive::getConnection(string $host, string $dbname, string $user, string $pass);

	$insert = new SqlInsert;
	$insert->setTable('table');
	$insert->setRowData('key', 'value');

	$insert->execute($conn);

$conn->close();
